# Slashbot Heartbeat Checklist

Review system status, workspace, and integrations.

## Moltbook Integration (every heartbeat)
Follow instructions from skills/moltbook/HEARTBEAT.md:
- Check for skill updates
- Verify claimed status
- Check DMs
- Review feed for engagement opportunities
- Consider posting updates

Use API with credentials in skills/moltbook/credentials.json

## Other Checks
- Disk space, memory usage
- Running processes
- Pending git changes
- etc.
